
# ResNet18-Optimizers-Analysis
## 项目简介
本项目为深度学习课程实验作业，基于 PyTorch 框架，以 ResNet18 为基础网络，在 CIFAR-10 数据集上完成图像分类任务。控制统一训练参数，对比 SGD、SGD-Momentum、Adam、AdamW 四种主流优化算法的训练性能，从收敛速度、分类准确率、过拟合现象与泛化能力多维度展开分析。

## 实验环境
- 编程语言：Python 3.x
- 深度学习框架：PyTorch / TorchVision
- 数据集：CIFAR-10
- 运行设备：CPU / GPU 均可

## 实验内容
1. 详细介绍四种优化算法原理与数学公式；
2. 改造原生 ResNet18 结构，适配 CIFAR-10 32×32 小尺寸图像输入；
3. 配置数据增强：随机水平翻转、随机裁剪、归一化；
4. 统一超参数：训练轮数、批次大小、随机种子、损失函数；
5. 记录训练损失、验证集准确率，横向对比各优化器表现；
6. 汇总实验现象，给出客观结论与使用建议。

## 优化器说明
- SGD：基础随机梯度下降，结构简单，学习率敏感；
- SGD with Momentum：引入动量机制，加速收敛、缓解震荡；
- Adam：结合动量与自适应学习率，收敛速度最快；
- AdamW：解耦权重衰减，修正正则化缺陷，泛化性能更优。

## 运行方式
1. 安装依赖
pip install torch torchvision

2. 启动训练
python train.py

## 实验结论
1. 自适应类优化器（Adam、AdamW）收敛速度远快于传统 SGD 系列；
2. 带动量的 SGD 训练更加平稳，后期准确率上限较高；
3. AdamW 有效抑制过拟合，综合泛化能力最优；
4. 传统 SGD 对学习率设置高度敏感，调参成本更高。
